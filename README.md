# Brand Management System

## Project Description
This project is a Java-based Brand Management System developed using Object-Oriented Programming (OOP) principles.

The system allows managing brands and products through a structured and modular architecture. It was developed as part of a university project to improve software development and system design skills.

## Technologies
- Java
- Object-Oriented Programming (OOP)

## Features
- Add, delete, and update brands
- Product management operations
- Organized class-based system structure
